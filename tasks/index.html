    Build Tasks¶

    Tasks are grouped into phases. Each task has its own file. Work through them in order — later tasks depend on earlier ones.

    Phases¶

    Phase 1 — Foundation (no AI, no Drive yet)¶

    Task File Description
    T01 t01_scaffolding.md Create folder structure, venv, install deps
    T02 t02_neo4j_setup.md Neo4j Aura account + connection test
    T03 t03_fastapi_skeleton.md FastAPI app boots, health endpoint works
    T04 t04_neo4j_schema.md Create constraints + indexes in Neo4j

    Phase 2 — Core data (papers, people, tags, projects)¶

    Task File Description
    T05 t05_paper_crud.md Create / read / list papers in Neo4j
    T06 t06_person_crud.md Create / read people + SPECIALIZES_IN
    T07 t07_tags_topics.md Tags and Topics CRUD + link to papers
    T08 t08_projects.md Projects CRUD + IN_PROJECT + RELATED_TO
    T09 t09_notes.md Notes CRUD + @/# mention parser

    Phase 3 — File handling + AI¶

    Task File Description
    T10 t10_pdf_parser.md Extract text + metadata from uploaded PDF
    T11 t11_google_drive.md Upload PDF to Drive, store drive_file_id
    T12 t12_claude_summary.md Auto-summarize paper text via Claude
    T13 t13_paper_ingest.md Full ingest pipeline: drop PDF → everything
    T14 t14_chat.md Chat with a paper via Claude

    Phase 4 — Search¶

    Task File Description
    T15 t15_search.md Full-text search across title, summary, notes

    Phase 5 — Frontend¶

    Task File Description
    T16 t16_frontend_setup.md React + Vite + Tailwind, boots locally
    T17 t17_library_page.md Paper grid + drag & drop upload
    T18 t18_paper_detail.md Paper detail page + note editor
    T19 t19_filters.md Filter by tag, topic, project, person
    T20 t20_chat_ui.md Chat panel in paper detail
    T21 t21_people_projects.md People and Projects pages

    Phase 6 — MCP Server¶

    Task File Description
    T22 t22_mcp_server.md MCP server: expose all tools to Claude Code
